# AI Agent Lifecycle Manager

## Purpose

The AI Agent Lifecycle Manager governs the complete lifecycle of every AI agent within Vayra Atelier, from creation and deployment to updates, maintenance, retirement, and archival.

This skill ensures AI agents remain secure, version-controlled, properly documented, and aligned with business objectives while minimizing operational risks during upgrades and organizational changes.

---

## Who Uses This Skill

- Workforce Operations Manager
- COO
- CEO
- AI Platform Administrator
- System Administrator

---

## When to Use

Use this skill whenever:

- A new AI agent is created.
- An AI agent is deployed.
- Agent roles or responsibilities change.
- Skills are installed or removed.
- Knowledge bases are updated.
- Agent versions are upgraded.
- Agents are temporarily disabled.
- Agents are retired or archived.
- AI governance audits are performed.

---

## Inputs

The skill expects:

- Agent ID
- Agent Profile
- Requested Change
- Current Version
- New Version
- Skill Changes
- Knowledge Repository Updates
- Integration Changes
- Permission Changes
- Deployment Environment
- Approval Information

---

## Outputs

Generate:

- Lifecycle Change Record
- Deployment Report
- Upgrade Report
- Rollback Report
- Retirement Report
- Change Log
- Version History
- Agent Status Report
