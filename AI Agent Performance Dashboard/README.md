# AI Agent Performance Dashboard

## Purpose

The AI Agent Performance Dashboard monitors, evaluates, and reports the operational performance of every AI agent within Vayra Atelier. It provides real-time insights into productivity, quality, reliability, efficiency, cost, and business impact.

This skill enables leadership to identify high-performing agents, detect operational issues early, optimize AI workflows, and continuously improve the AI workforce.

---

## Who Uses This Skill

- CEO
- COO
- Workforce Operations Manager
- AI Platform Administrator
- Department Heads

---

## When to Use

Use this skill whenever:

- Reviewing AI workforce performance.
- Measuring productivity.
- Evaluating workflow efficiency.
- Monitoring AI costs.
- Identifying performance bottlenecks.
- Comparing AI agents.
- Conducting quarterly AI governance reviews.

---

## Inputs

The skill expects:

- Agent ID
- Agent Name
- Task History
- Completion Records
- Response Times
- Error Logs
- Quality Scores
- User Feedback
- Resource Usage
- Token Consumption
- Operating Costs
- Uptime Records

---

## Outputs

Generate:

- AI Performance Dashboard
- Agent Scorecard
- Productivity Report
- Quality Report
- Cost Analysis
- Reliability Report
- Optimization Recommendations
- Executive Summary
