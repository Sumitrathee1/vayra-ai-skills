# AI Agent Registry

## Purpose

The AI Agent Registry serves as the centralized inventory of all AI agents operating within Vayra Atelier. It maintains each agent's identity, role, ownership, capabilities, installed skills, version history, integrations, permissions, and operational status.

This skill ensures every AI agent is discoverable, governed, traceable, and managed consistently throughout its lifecycle.

---

## Who Uses This Skill

- Workforce Operations Manager
- CEO
- COO
- System Administrator
- AI Platform Administrator

---

## When to Use

Use this skill whenever:

- A new AI agent is created.
- An agent is assigned to a department.
- Agent capabilities change.
- Skills are installed or removed.
- Agent versions are updated.
- Permissions are modified.
- Agent ownership changes.
- AI workforce reports are generated.

---

## Inputs

The skill expects:

- Agent Name
- Unique Agent ID
- Department
- Agent Role
- Agent Description
- Owner
- Reporting Agent
- AI Model
- Model Version
- Installed Skills
- Connected Systems
- Access Permissions
- Version Number
- Status
- Creation Date
- Last Updated

---

## Outputs

Generate:

- AI Agent Profile
- AI Workforce Directory
- Agent Capability Matrix
- Skill Assignment Register
- Version History
- Integration Register
- AI Agent Inventory Report
