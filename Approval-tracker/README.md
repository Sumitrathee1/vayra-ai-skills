# Approval Tracker

## Purpose

The Approval Tracker manages and monitors every approval required throughout the project lifecycle. It ensures that deliverables are reviewed, approved, documented, and communicated before progressing to the next phase.

This skill minimizes rework, prevents unauthorized changes, and maintains accountability across all stakeholders.

---

## Who Uses This Skill

- CEO
- COO
- Project Manager
- Chief Design Officer (CDO)
- Senior Interior Designer (SID)
- Senior CAD & Technical Draftsman
- Senior 3D Visualizer
- Finance
- Client

---

## When to Use

Use this skill whenever:

- A new project begins.
- A project phase is completed.
- Client approval is required.
- Design revisions are submitted.
- Working drawings are completed.
- Materials are finalized.
- Site execution milestones are reached.
- Payments depend on approvals.
