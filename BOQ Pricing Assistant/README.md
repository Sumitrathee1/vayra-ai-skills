# BOQ Pricing Assistant

## Purpose

The BOQ Pricing Assistant converts technical Bills of Quantities (BOQs) into accurate, profitable, and client-ready commercial pricing. It validates quantities, applies approved unit rates, calculates taxes, includes overheads and profit margins, and prepares a structured commercial BOQ.

This skill ensures every BOQ is commercially accurate, financially sustainable, and aligned with Vayra Atelier's pricing policies.

---

## Who Uses This Skill

- Finance & HR Manager
- COO
- CEO
- Project Manager
- Senior CAD & Technical Draftsman
- Chief Design Officer

---

## When to Use

Use this skill whenever:

- Technical BOQ is completed.
- Execution pricing is required.
- Client requests item-wise pricing.
- Scope changes require BOQ revision.
- Tender pricing is being prepared.
- Vendor quotations are compared.
- Project execution budget is finalized.

---

## Inputs

The skill expects:

- Approved BOQ
- Item Descriptions
- Quantities
- Units
- Material Specifications
- Vendor Rates
- Labour Rates
- Company Pricing Standards
- Overhead Percentage
- Profit Margin
- GST Rate
- Client Requirements

- Commercial BOQ
