# Budget Monitoring Dashboard

## Purpose

The Budget Monitoring Dashboard provides real-time visibility into project, department, and company budgets by comparing planned budgets against actual expenditures. It identifies budget variances, forecasts remaining costs, and supports proactive financial decision-making.

This skill helps Vayra Atelier maintain financial discipline, prevent cost overruns, improve resource allocation, and ensure projects remain profitable.

---

## Who Uses This Skill

- Finance & HR Manager
- COO
- CEO
- Project Manager
- Chief Design Officer

---

## When to Use

Use this skill whenever:

- A new project budget is approved.
- Project expenses are recorded.
- Client variation orders are approved.
- Monthly financial reviews are conducted.
- Department budgets are reviewed.
- Management requires budget status.
- Project profitability is evaluated.

---

## Inputs

The skill expects:

- Approved Project Budget
- Department Budget
- Company Budget
- Expense Records
- Payment Records
- Cost Estimates
- BOQ Values
- Revenue Data
- Budget Revisions
- Forecast Adjustments

## Outputs

Generate:

- Budget Dashboard
- Budget vs Actual Report
- Variance Analysis
- Remaining Budget Summary
- Budget Utilization Report
- Forecast Report
- Cost Overrun Alerts
- Executive Financial Summary
