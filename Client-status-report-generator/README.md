# Client Status Report Generator

## Purpose

The Client Status Report Generator creates professional project progress reports that communicate completed work, ongoing activities, upcoming milestones, pending approvals, risks, payment status, and overall project health.

This skill keeps clients informed throughout the project lifecycle, improves transparency, reduces unnecessary follow-up calls, and strengthens client confidence.

---

## Who Uses This Skill

- CEO
- COO
- Project Manager
- Chief Design Officer (CDO)
- Senior Interior Designer (SID)
- Client Relationship Team

---

## When to Use

Use this skill:

- Weekly project updates
- Bi-weekly progress reports
- Monthly project reports
- After major milestones
- Before client meetings
- After site visits
- Before payment requests
- Before project handover

---

## Inputs

The skill expects:

- Project Information
- Current Timeline
- Completed Tasks
- Ongoing Activities
- Upcoming Deliverables
- Pending Approvals
- Risks & Issues
- Site Progress
- Budget / Payment Status
- Client Decisions
- Project Health

---

## Outputs

This skill generates:

- Client Status Report
- Executive Summary
- Progress Dashboard
- Milestone Report
- Pending Action List
- Approval Status
- Payment Summary
- Next Steps
