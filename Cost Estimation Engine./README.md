# Cost Estimation Engine

## Purpose

The Cost Estimation Engine calculates the estimated project cost for interior design and turnkey execution projects by analyzing project scope, materials, labour, execution complexity, design fees, overheads, contingencies, and profit margins.

This skill ensures every estimate is accurate, consistent, profitable, and aligned with Vayra Atelier's commercial standards before preparing quotations or BOQs.

---

## Who Uses This Skill

- Finance & HR Manager
- COO
- CEO
- Project Manager
- Chief Design Officer

---

## When to Use

Use this skill whenever:

- A new enquiry is received.
- A quotation is being prepared.
- BOQ pricing is required.
- Scope changes occur.
- Client requests additional work.
- Budget revisions are required.
- Value engineering is performed.

---

## Inputs

The skill expects:

- Client Information
- Project Type
- Total Area
- Scope of Work
- Material Specifications
- Labour Requirements
- Design Fees
- Site Location
- Execution Complexity
- Vendor Pricing
- Transportation Costs
- Overheads
- Contingency Percentage
- Target Profit Margin

---

## Outputs

Generate:

- Project Cost Estimate
- Material Cost Summary
- Labour Cost Summary
- Design Cost
- Execution Cost
- Overhead Cost
- Contingency Cost
- Profit Margin
- Final Estimated Project Cost
