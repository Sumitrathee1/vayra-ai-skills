# Employee Records Manager

## Purpose

The Employee Records Manager serves as the centralized repository for all employee information within Vayra Atelier. It maintains complete employment records, contractual information, professional qualifications, assigned roles, access permissions, company assets, and employment history throughout the employee lifecycle.

This skill ensures employee information remains secure, accurate, up-to-date, and accessible to authorized leadership while supporting payroll, performance management, compliance, and workforce planning.

---

## Who Uses This Skill

- Workforce Operations Manager
- CEO
- COO

---

## When to Use

Use this skill whenever:

- A new employee joins.
- Employee information changes.
- Employment documents are updated.
- Roles or responsibilities change.
- Company assets are assigned.
- Employee records require verification.
- Management reviews workforce information.

---

## Inputs

The skill expects:

- Employee Information
- Employment Details
- Contact Information
- Emergency Contact
- Contract Information
- Designation
- Department
- Reporting Manager
- Skills
- Certifications
- Company Assets
- Access Permissions
- Employment Status

---

## Outputs

Generate:

- Employee Profile
- Employee Master Record
- Employment Summary
- Skills Register
- Asset Register
- Workforce Directory
- Employee Status Report
