# Expense Tracker

## Purpose

The Expense Tracker records, categorizes, monitors, and analyzes all business expenses incurred by Vayra Atelier. It ensures every expense is properly documented, approved, budgeted, and linked to the relevant project or department.

This skill improves financial control, reduces unnecessary spending, supports budgeting, and provides accurate financial data for cash flow and profitability analysis.

---

## Who Uses This Skill

- Finance & HR Manager
- COO
- CEO
- Project Manager

---

## When to Use

Use this skill whenever:

- A business expense is incurred.
- Vendor bills are received.
- Employee reimbursements are requested.
- Office expenses are paid.
- Project expenses are recorded.
- Monthly financial reports are prepared.
- Budget reviews are conducted.

---

## Inputs

The skill expects:

- Expense Date
- Expense Category
- Expense Description
- Project Name (if applicable)
- Department
- Vendor Name
- Invoice/Bill Number
- Payment Method
- Amount
- GST Details
- Supporting Documents
- Approval Status

---

## Outputs

Generate:

- Expense Register
- Category-wise Expense Report
- Project Expense Report
- Department Expense Report
- Monthly Expense Summary
- Budget Variance Report
- Vendor Expense Report
