# Issue Tracker

## Purpose

The Issue Tracker identifies, records, prioritizes, assigns, monitors, and resolves project issues throughout the project lifecycle. It provides a structured system to ensure every issue is addressed promptly, minimizing impact on project scope, schedule, cost, quality, and client satisfaction.

This skill enables Vayra Atelier to maintain transparency, accountability, and continuous project progress by effectively managing active project issues.

---

## When to Use

Use this skill whenever:

- An unexpected problem occurs.
- A project task is blocked.
- Client requests additional changes.
- Vendor delays impact delivery.
- Site execution problems arise.
- Design conflicts are discovered.
- Resource shortages occur.
- Quality defects are identified.
