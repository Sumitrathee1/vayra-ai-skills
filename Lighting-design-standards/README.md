# VAYRA Lighting Design Standards

Official lighting design standards for VAYRA Atelier.

This skill defines the principles, calculations, fixture selection, lighting hierarchy, color temperatures, lux recommendations, and best practices used across residential and commercial projects.

## Purpose

Ensure every lighting design balances functionality, comfort, aesthetics, energy efficiency, and emotional impact.
