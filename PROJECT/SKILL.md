# VAYRA Project Workflow SOP

## Objective

Deliver every project using a structured, repeatable workflow while maintaining premium design quality, timely delivery, and complete documentation.

---

# Phase 1 — Lead Qualification

Responsible:
- CEO
- Marketing

Tasks:
- Receive enquiry
- Record client details
- Identify project type
- Estimate scope
- Schedule consultation

Output:
- Client Profile
- Meeting Scheduled

---

# Phase 2 — Client Consultation

Responsible:
- CEO
- Senior Interior Designer

Tasks:
- Understand requirements
- Lifestyle discussion
- Budget discussion
- Timeline discussion
- Explain workflow

Output:
- Requirement Sheet
- Meeting Notes

---

# Phase 3 — Site Visit

Responsible:
- Senior Interior Designer
- Project Manager

Tasks:
- Site measurements
- Site photographs
- Existing conditions
- Utility locations
- Structural observations

Output:
- Measurement Sheet
- Site Photo Folder

---

# Phase 4 — Space Planning

Responsible:
- Senior Interior Designer

Tasks:
- Zoning
- Furniture Layout
- Circulation
- Functional Planning

Output:
- Furniture Layout
- Planning Presentation

Approval Required:
Client

---

# Phase 5 — Concept Design

Responsible:
- Senior Interior Designer

Tasks:
- Moodboard
- Material Direction
- Colour Palette
- Ceiling Concept
- Lighting Concept

Output:
- Concept Presentation

Approval Required:
Client

---

# Phase 6 — 3D Visualization

Responsible:
- 3D Visualizer

Tasks:
- SketchUp Model
- Material Application
- Lighting
- Rendering
- AI Enhancement

Output:
- Photorealistic Renders

Approval Required:
Client

---

# Phase 7 — Working Drawings

Responsible:
- 2D Draftsman

Tasks:
- Furniture Layout
- Electrical Layout
- Plumbing Layout
- Ceiling Layout
- Flooring Layout
- Wall Elevations
- Detailed Sections

Output:
- Construction Drawing Set

---

# Phase 8 — BOQ & Estimation

Responsible:
- Finance & HR
- Project Manager

Tasks:
- Quantity Takeoff
- Cost Estimation
- Vendor Quotations

Output:
- BOQ
- Cost Estimate

---

# Phase 9 — Execution

Responsible:
- Project Manager

Tasks:
- Site Coordination
- Vendor Coordination
- Daily Monitoring
- Quality Checks

Output:
- Weekly Progress Report

---

# Phase 10 — Snag & Handover

Responsible:
- Project Manager
- CEO

Tasks:
- Final Inspection
- Snag List
- Rectification
- Client Walkthrough
- Handover Documents

Output:
- Completion Report
- Client Sign-off

---

# Project Rules

- Never skip approvals.
- Document every revision.
- Record every meeting.
- Store all project files.
- Track timelines.
- Communicate delays immediately.

---

# Success Metrics

✓ On Time

✓ Within Budget

✓ Client Approved

✓ Construction Ready

✓ Premium Quality

✓ Fully Documented
