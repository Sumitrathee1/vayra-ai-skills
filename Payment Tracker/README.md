# Payment Tracker

## Purpose

The Payment Tracker monitors and manages all client payments throughout the project lifecycle. It records invoices, received payments, outstanding balances, overdue amounts, payment milestones, and collection status while providing real-time visibility into accounts receivable.

This skill helps Vayra Atelier maintain healthy cash flow, improve collection efficiency, and reduce payment disputes through systematic tracking and reporting.

---

## Who Uses This Skill

- Finance & HR Manager
- COO
- CEO
- Project Manager

---

## When to Use

Use this skill whenever:

- A new invoice is issued.
- A client payment is received.
- Payment reminders are required.
- Outstanding balances need review.
- Overdue invoices must be followed up.
- Financial reports are prepared.
- Project payment status is reviewed.

---

## Inputs

The skill expects:

- Client Information
- Project Information
- Invoice Details
- Payment Schedule
- Invoice Amount
- Payment Received
- Payment Date
- Payment Method
- Bank Reference
- Outstanding Balance
- Due Date

---

## Outputs

Generate:

- Payment Status
- Outstanding Balance Report
- Accounts Receivable Report
- Overdue Payment Report
- Payment History
- Collection Summary
- Client Ledger
