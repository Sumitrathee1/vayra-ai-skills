# Payroll Coordinator

## Purpose

The Payroll Coordinator manages the complete payroll lifecycle for Vayra Atelier by calculating employee compensation, verifying payroll data, applying statutory deductions, processing reimbursements, and generating payroll reports.

This skill ensures payroll is accurate, compliant, confidential, and processed on time while integrating with employee records, performance incentives, and financial systems.

---

## Who Uses This Skill

- Workforce Operations Manager
- Finance & HR Manager
- COO
- CEO

---

## When to Use

Use this skill whenever:

- Monthly payroll is processed.
- New employees join.
- Employee salaries are revised.
- Incentives or bonuses are approved.
- Reimbursements are claimed.
- Final settlements are prepared.
- Payroll reports are required.

---

## Inputs

The skill expects:

- Employee Records
- Salary Structure
- Incentive Data
- Reimbursement Claims
- Bonus Information
- Deduction Details
- Statutory Contributions
- Tax Information
- Payroll Period
- Final Settlement Details (if applicable)

---

## Outputs

Generate:

- Monthly Payroll Register
- Employee Salary Slip
- Payroll Summary
- Deduction Report
- Reimbursement Report
- Final Settlement Report
- Payroll Approval Report
