# Performance Review Manager

## Purpose

The Performance Review Manager evaluates employee performance through structured, objective, and data-driven assessments. It measures individual contributions, project outcomes, technical skills, collaboration, leadership potential, and alignment with Vayra Atelier's standards.

This skill provides actionable feedback, identifies development opportunities, supports promotion decisions, and encourages continuous improvement across the organization.

---

## Who Uses This Skill

- Workforce Operations Manager
- CEO
- COO
- Department Heads
- Team Leads

---

## When to Use

Use this skill whenever:

- Quarterly performance reviews are conducted.
- Annual appraisals are prepared.
- Employee probation is reviewed.
- Promotion decisions are evaluated.
- Salary revisions are considered.
- Performance improvement plans are required.
- Leadership potential is assessed.

---

## Inputs

The skill expects:

- Employee Profile
- Role & Responsibilities
- Project Assignments
- KPI Results
- Project Delivery Records
- Client Feedback
- Peer Feedback
- Manager Feedback
- Quality Review
- Training Records
- Previous Review History

---

## Outputs

Generate:

- Performance Review Report
- KPI Scorecard
- Competency Assessment
- Strength Analysis
- Improvement Areas
- Development Plan
- Promotion Recommendation
- Performance Summary
