# Profitability Analyzer

## Purpose

The Profitability Analyzer evaluates the financial performance of projects, clients, services, and departments by comparing revenue against all direct and indirect costs. It measures gross profit, net profit, profit margins, and return on investment (ROI) while identifying opportunities to improve profitability.

This skill enables Vayra Atelier to make data-driven business decisions, optimize pricing strategies, improve operational efficiency, and maximize long-term profitability.

## When to Use

Use this skill whenever:

- A project reaches a billing milestone.
- A project is completed.
- Monthly financial reports are prepared.
- Quarterly business reviews are conducted.
- Pricing strategies are evaluated.
- Management reviews company performance.
- Client profitability is analyzed.
