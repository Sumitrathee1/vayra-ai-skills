# Project Timeline Builder

## Purpose

The Project Timeline Builder converts a Work Breakdown Structure (WBS) into a structured project schedule with realistic durations, dependencies, milestones, approval gates, and delivery dates.

This skill ensures every Vayra Atelier project follows a predictable execution timeline from client onboarding to final handover.

---

## Who Uses This Skill

- COO
- Project Manager
- CEO (Project Monitoring)
- Senior Interior Designer (Design Scheduling)

---

## When to Use

Use this skill whenever:

- A new project begins.
- A project schedule must be created.
- Scope changes affect timelines.
- Client deadlines change.
- Resource allocation needs adjustment.
- A delayed project requires replanning.

---

## Inputs

The skill expects:

- Work Breakdown Structure
- Task List
- Estimated Durations
- Dependencies
- Team Availability
- Client Deadline
- Project Constraints
- Approval Requirements

---

## Outputs

This skill generates:

- Project Timeline
- Phase Schedule
- Milestone Plan
- Critical Path
- Approval Timeline
- Resource Schedule
- Buffer Planning
- Client Delivery Schedule
