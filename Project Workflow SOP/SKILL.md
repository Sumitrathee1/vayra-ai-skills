# VAYRA Project Workflow SOP

## Purpose

Provide a repeatable, organized project delivery process for every VAYRA Atelier project.

---

# Phase 1 — Client Enquiry

Objectives

- Understand client requirements
- Record contact information
- Identify project type
- Estimate project scope
- Schedule consultation

Deliverables

- Client Information Sheet
- Meeting Schedule

Responsible

CEO / Marketing

---

# Phase 2 — Initial Consultation

Objectives

- Understand lifestyle
- Understand business goals
- Discuss budget
- Discuss timeline
- Explain company workflow

Deliverables

- Meeting Notes
- Requirement Sheet

Responsible

CEO / SID

---

# Phase 3 — Site Visit

Objectives

- Site measurements
- Site photographs
- Existing conditions
- Utilities check
- Structural observations

Deliverables

- Site Measurement File
- Photo Documentation

Responsible

SID / Project Manager

---

# Phase 4 — Concept Design

Tasks

- Space planning
- Mood board
- Material direction
- Design concept
- Furniture layout

Deliverables

- Concept Presentation
- Furniture Layout

Responsible

SID

Approval Required

Client

---

# Phase 5 — Design Development

Tasks

- Ceiling concept
- Material selection
- Lighting concept
- Furniture details
- Color palette

Deliverables

- Design Presentation
- Material Board

Responsible

SID

---

# Phase 6 — 3D Visualization

Tasks

- SketchUp model
- Render setup
- AI enhancement
- Photoshop post-production

Deliverables

- Photorealistic Renders

Responsible

3D Visualizer

---

# Phase 7 — Working Drawings

Tasks

- Furniture layout
- Electrical layout
- Plumbing layout
- Ceiling layout
- Flooring layout
- Wall elevations
- Detail drawings

Deliverables

- Construction Drawing Set

Responsible

2D Draftsman

---

# Phase 8 — Client Approval

Required Approvals

- Layout
- Materials
- Renders
- Drawings
- Budget

Responsible

Project Manager

---

# Phase 9 — BOQ & Procurement

Tasks

- Quantity takeoff
- Vendor quotations
- Cost verification

Responsible

Finance & HR
Project Manager

---

# Phase 10 — Execution

Tasks

- Site supervision
- Vendor coordination
- Quality inspection
- Progress monitoring

Responsible

Project Manager

---

# Phase 11 — Snag List

Tasks

- Final inspection
- Defect identification
- Rectification tracking

Responsible

Project Manager

---

# Phase 12 — Project Handover

Deliverables

- Final photographs
- Completion report
- Warranty documents
- Client feedback

Responsible

CEO
Project Manager

---

# Project Rules

Never skip approvals.

Document every change.

Record every meeting.

Track every revision.

Maintain organized project folders.

Communicate delays immediately.

---

# Success Criteria

Project delivered:

✓ On Time

✓ Within Budget

✓ Approved by Client

✓ Construction Ready

✓ Premium Quality

✓ Properly Documented
