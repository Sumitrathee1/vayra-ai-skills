# Project Health Dashboard

## Purpose

The Project Health Dashboard provides a comprehensive overview of a project's overall performance by monitoring schedule, budget, quality, resources, risks, approvals, issues, and client satisfaction.

This skill enables Vayra Atelier to proactively identify project concerns, support data-driven decision-making, and maintain successful project delivery.

- COO
- Project Manager
- Chief Design Officer (CDO)
- Senior Interior Designer (SID)
- Finance

## When to Use

Use this skill:

- Weekly Project Reviews
- Monthly Management Reviews
- Leadership Meetings
- Client Progress Reviews
- Project Milestone Reviews
- Before Project Handover
- Portfolio Performance Reviews
