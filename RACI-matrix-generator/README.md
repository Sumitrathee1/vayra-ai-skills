# RACI Matrix Generator

## Purpose

The RACI Matrix Generator defines clear roles and responsibilities for every task, deliverable, milestone, and decision within a project. It eliminates ambiguity by assigning accountability and communication responsibilities across the project team.

This skill ensures that every project follows a structured responsibility framework, improving coordination, reducing delays, and increasing accountability.

---

## When to Use

Use this skill whenever:

- A new project starts.
- Tasks are assigned to the team.
- Team responsibilities change.
- Scope is revised.
- New stakeholders are introduced.
- Cross-functional collaboration is required.

---

## Inputs

The skill expects:

- Work Breakdown Structure
- Task List
- Project Timeline
- Team Structure
- Department Responsibilities
- Approval Hierarchy

---

## Outputs

This skill generates:

- RACI Matrix
- Responsibility Assignment Table
- Stakeholder Communication Matrix
- Accountability Map
- Review & Approval Roles
