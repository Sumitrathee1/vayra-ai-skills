# Project Planning SOP

## Purpose

The Project Planning SOP helps AI agents create structured, executable project plans for interior design projects at Vayra Atelier.

This skill converts a client brief into a complete execution roadmap including phases, deliverables, ownership, timelines, dependencies, approval gates, and project risks.

---

## Who Uses This Skill

- COO
- Project Manager
- CEO (for review)
- Senior Interior Designer (when planning design phases)

---

## When to Use

Use this skill whenever:

- A new project starts.
- A project enters a new phase.
- Client scope changes.
- A delayed project requires replanning.
- Major revisions affect the schedule.

---

## Inputs

The skill expects:

- Client Brief
- Site Information
- Scope of Work
- Budget
- Timeline
- Team Availability
- Constraints
- Existing Drawings (if available)

---

## Outputs

This skill generates:

- Project Roadmap
- Work Breakdown Structure
- Phase Plan
- Milestones
- Responsibility Assignment
- Timeline
- Dependency Map
- Risk Register
- Approval Gates
