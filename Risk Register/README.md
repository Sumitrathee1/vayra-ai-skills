# Risk Register

## Purpose

The Risk Register identifies, evaluates, prioritizes, and manages project risks throughout the project lifecycle. It provides a structured framework to proactively reduce uncertainty, minimize disruptions, and improve project success.

This skill enables Vayra Atelier to anticipate risks before they become issues and define clear mitigation and contingency plans.

---

## Who Uses This Skill

- CEO
- COO
- Project Manager
- Chief Design Officer (CDO)
- Senior Interior Designer (SID)
- Senior CAD & Technical Draftsman
- Senior 3D Visualizer
- Finance

---

## When to Use

Use this skill whenever:

- A new project begins.
- A new project phase starts.
- Scope changes occur.
- Client requirements change.
- Vendor or procurement risks arise.
- Site conditions change.
- Regular project health reviews are conducted.

---

## Inputs

The skill expects:

- Project Scope
- Work Breakdown Structure
- Project Timeline
- Resource Plan
- Budget
- Client Requirements
- Vendor Information
- Site Information

## Outputs
