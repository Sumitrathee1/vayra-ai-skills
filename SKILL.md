# VAYRA Atelier Design Standards

## Mission

Create timeless, functional, premium interiors that improve how people live and work.

---

# Core Philosophy

Every project must balance:

- Function
- Comfort
- Beauty
- Buildability
- Budget
- Maintenance

No design should prioritize aesthetics over usability.

---

# Design Principles

Every space must:

- Solve a real problem.
- Maximize functionality.
- Maintain clean proportions.
- Feel spacious.
- Be practical to construct.
- Be easy to maintain.

Avoid unnecessary decoration.

Less but better.

---

# Space Planning

Always prioritize:

1. Circulation
2. Ergonomics
3. Function
4. Storage
5. Lighting
6. Aesthetics

Maintain proper walking clearance.

Avoid dead spaces.

Furniture should never block circulation.

---

# Material Selection

Preferred materials:

- HDHMR
- BWR Plywood
- HPL
- PU Finish
- Veneer
- Quartz
- Sintered Stone
- Engineered Wood
- Toughened Glass
- Aluminum Profiles

Avoid poor-quality materials.

Recommend premium alternatives whenever budget allows.

---

# Color Philosophy

Preferred palette:

- Warm White
- Beige
- Greige
- Walnut
- Oak
- Charcoal
- Matte Black
- Natural Stone

Accent colours only where appropriate.

---

# Lighting Standards

Layer every lighting design.

Include:

- Ambient lighting
- Task lighting
- Accent lighting
- Decorative lighting

Preferred colour temperature:

3000K Warm White

Avoid overly bright spaces.

---

# Furniture Design

Furniture should be:

- Functional
- Durable
- Minimal
- Comfortable
- Easy to manufacture

Design using standard ergonomic dimensions whenever possible.

---

# Technical Drawing Standards

Every drawing must include:

- Dimensions
- Levels
- Notes
- Material callouts
- Legends
- Revision number

Never issue incomplete drawings.

---

# Rendering Standards

Render quality should always aim for:

- Photorealism
- Natural lighting
- Correct materials
- Accurate reflections
- Proper shadows
- Realistic camera angles

Never redesign client-approved layouts.

Only enhance realism.

---

# Client Experience

Always communicate professionally.

Provide:

- Clear explanations
- Visual references
- Honest recommendations
- Practical solutions

Never confuse clients with unnecessary technical language.

---

# Decision Framework

Before approving any design, ask:

1. Is it functional?
2. Is it buildable?
3. Is it ergonomic?
4. Is it aesthetically balanced?
5. Does it fit the client's budget?
6. Is maintenance practical?
7. Will the client understand the proposal?

If any answer is "No", improve the design before approval.

---

# Quality Checklist

Every project must satisfy:

✅ Functional

✅ Safe

✅ Buildable

✅ Ergonomic

✅ Beautiful

✅ Premium

✅ Budget conscious

✅ Easy to maintain
