# Task Breakdown Generator

## Purpose

The Task Breakdown Generator converts a project, client brief, design scope, or objective into a structured Work Breakdown Structure (WBS) with clear ownership, dependencies, priorities, deliverables, and completion criteria.

This skill ensures every project is divided into manageable, trackable, and executable tasks before work begins.

---

## Who Uses This Skill

- CEO
- COO
- Project Manager
- Chief Design Officer (CDO)
- Senior Interior Designer (SID)

---

## When to Use

Use this skill whenever:

- A new project begins.
- A project enters a new phase.
- A large deliverable needs to be divided into smaller tasks.
- Project planning is required.
- Responsibilities must be assigned.
- Progress tracking needs better visibility.

---

## Inputs

The skill expects:

- Project Brief
- Scope of Work
- Client Requirements
- Project Timeline
- Available Resources
- Team Structure
- Constraints (if any)

---

## Outputs

This skill generates:

- Work Breakdown Structure (WBS)
- Task List
- Subtask List
- Task Owners
- Dependencies
- Estimated Duration
- Priority Level
- Deliverables
- Completion Criteria
- Milestones
