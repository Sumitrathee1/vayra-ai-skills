# Workforce Dashboard

## Purpose

The Workforce Dashboard serves as the executive command center for monitoring the complete workforce of Vayra Atelier, including both human employees and AI agents. It consolidates workforce data, operational metrics, productivity indicators, resource allocation, organizational health, and strategic insights into a single dashboard.

This skill enables leadership to make informed decisions, monitor organizational performance, optimize workforce utilization, and maintain operational excellence.

---

## Who Uses This Skill

- CEO
- COO
- Workforce Operations Manager
- Department Heads
- Project Manager

---

## When to Use

Use this skill whenever:

- Reviewing daily operations.
- Monitoring workforce productivity.
- Evaluating department performance.
- Planning resource allocation.
- Reviewing AI and human collaboration.
- Conducting executive meetings.
- Measuring organizational performance.
