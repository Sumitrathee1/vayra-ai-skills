# VAYRA AutoCAD Drafting Standards

Official AutoCAD drafting standards for VAYRA Atelier.

This skill defines drawing organization, layer management, annotation, dimensioning, plotting, and documentation standards to ensure every drawing package is professional, coordinated, and construction-ready.

## Purpose

Create consistent, readable, and accurate working drawings that reduce site errors and improve execution quality.

## Covers

- Drawing Setup
- Layer Standards
- Text Styles
- Dimensioning
- Lineweights
- Hatching
- Blocks
- Sheet Layout
- Plotting
- Revision Control
- Quality Checklist
