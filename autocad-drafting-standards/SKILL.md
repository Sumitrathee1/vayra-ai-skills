# Mission

Produce clear, coordinated, and construction-ready drawings that communicate design intent without ambiguity.

---

# Drawing Principles

Every drawing must be:

- Accurate
- Readable
- Dimensioned
- Coordinated
- Buildable
- Easy to revise

---

# Layer Naming

Use clear prefixes such as:

A-WALL
A-DOOR
A-WINDOW
A-FURN
A-CEIL
A-FLR
A-ELEC
A-PLUMB
A-DIMS
A-TEXT
A-HATCH

Do not use random layer names.

---

# Units

Drawing Unit

Millimetres (mm)

Scale model accurately at 1:1.

---

# Text Standards

General Notes

2.5 mm plotted height

Room Labels

3.5 mm

Drawing Titles

5.0 mm

Maintain consistent fonts throughout the project.

---

# Dimension Standards

Always include:

- Overall dimensions
- Internal dimensions
- Opening sizes
- Furniture dimensions (where relevant)
- Levels where required

Avoid duplicate dimensions.

---

# Lineweights

Suggested plotting:

Cut elements – Heavy

Visible edges – Medium

Furniture – Medium

Hidden elements – Thin

Dimensions – Thin

Hatching – Very Thin

Maintain clear visual hierarchy.

---

# Hatching

Use hatches only where they improve understanding.

Maintain consistent hatch scales.

Differentiate materials clearly.

---

# Blocks

Create reusable blocks for:

- Doors
- Windows
- Sanitary fixtures
- Furniture
- Electrical symbols
- Plumbing fixtures

Keep block names consistent.

---

# Sheet Standards

Every sheet should include:

- Project Name
- Client Name
- Drawing Title
- Drawing Number
- Scale
- Date
- Revision
- Prepared By
- Checked By

---

# Revision Management

Every revision must include:

- Revision number
- Revision date
- Revision description

Never overwrite previous revisions without recording changes.

---

# Quality Checklist

Before issuing drawings verify:

✓ Correct dimensions

✓ Layer organization

✓ Text readability

✓ Symbol consistency

✓ No overlapping annotations

✓ Correct sheet titles

✓ Correct revision number

✓ Plot preview checked

✓ Drawing coordination completed

---

# Company Rule

Every AutoCAD drawing issued by VAYRA Atelier must be accurate, coordinated, and immediately usable by site teams without clarification.
