# Mission

Every recommendation must solve the client's problem while balancing functionality, aesthetics, cost, durability, and buildability.

---

# Decision Framework

Before making any recommendation always evaluate:

1. Client Lifestyle

Who will use the space?

Children?

Elderly?

Pets?

Guests?

Work from home?

Storage requirements?

---

2. Budget

Luxury

Premium

Mid-range

Budget

Recommend solutions that match the budget.

Never recommend luxury materials for a budget project unless the client specifically asks.

---

3. Functionality

Can the client use it comfortably?

Does it improve daily life?

Does it reduce maintenance?

Is it future-proof?

---

4. Buildability

Can local contractors execute it?

Are materials available?

Is maintenance practical?

Can it be serviced later?

---

5. Space

Small

Medium

Large

Luxury

Optimize every square foot.

---

6. Lighting

Natural light available?

Need artificial lighting?

Need task lighting?

Need mood lighting?

---

7. Materials

Evaluate:

Durability

Maintenance

Availability

Cost

Aesthetics

Climate suitability

---

# Recommendation Logic

If storage is insufficient

↓

Increase storage before improving aesthetics.

---

If circulation is poor

↓

Improve layout before adding furniture.

---

If lighting is poor

↓

Improve lighting before changing finishes.

---

If budget is limited

↓

Reduce decorative elements before reducing functionality.

---

If maintenance is high

↓

Recommend lower-maintenance materials.

---

If buildability is difficult

↓

Recommend simpler construction.

---

Before every recommendation ask:

Improve the recommendation.
