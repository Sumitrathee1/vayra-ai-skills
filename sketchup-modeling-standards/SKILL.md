# Mission

Create clean, organized, lightweight, and render-ready SketchUp models that are easy to edit, coordinate, and present.

---

# Modeling Principles

Every model must be:

- Organized
- Accurate
- Lightweight
- Scalable
- Render Ready
- Easy to Edit

---

# File Organization

Create folders for:

- CAD Imports
- Components
- Textures
- References
- Renders
- Exports

---

# Units

Millimetres (mm)

Always model at full scale (1:1).

---

# Groups & Components

Always convert repeated objects into Components.

Use Groups for unique geometry.

Never leave loose geometry.

---

# Naming Convention

Examples:

BED_MASTER

WARDROBE_B01

TV_UNIT_LIVING

SOFA_3_SEATER

KITCHEN_BASE_600

Use clear and consistent names.

---

# Tags

Recommended tags:

Architecture

Furniture

Ceiling

Lighting

Electrical

Plumbing

Decor

Landscape

Annotations

Construction

Do not draw directly on tagged geometry.

Keep raw geometry on Untagged.

---

# Scenes

Create scenes for:

- Overall View
- Living Room
- Bedroom
- Kitchen
- Bathroom
- Elevation
- Section
- Presentation Views

Use descriptive scene names.

---

# Materials

Use high-resolution textures.

Maintain correct texture scale.

Avoid duplicate materials.

Purge unused materials regularly.

---

# CAD Import

Before modeling:

- Clean CAD file
- Remove unnecessary layers
- Check units
- Align origin
- Delete duplicate lines

---

# Performance Optimization

Purge unused components.

Purge unused materials.

Reduce unnecessary polygon count.

Use proxies for heavy vegetation and furniture.

Avoid oversized texture files.

---

# Rendering Preparation

Verify:

- Face orientation
- Material mapping
- Camera scenes
- Lighting setup
- Hidden geometry removed

---

# Export Standards

Export:

- SKP
- DWG
- PDF (2D Views)
- FBX (if required)

Maintain consistent file naming.

---

# Quality Checklist

Before submission verify:

✓ Correct dimensions

✓ Clean Outliner

✓ Organized Tags

✓ No loose geometry

✓ Proper components

✓ Correct materials

✓ Optimized file size

✓ Scenes updated

✓ Purged unused items

✓ Render-ready model

---

# Common Mistakes

Never:

- Leave loose geometry.
- Explode components unnecessarily.
- Use random tag names.
- Import dirty CAD files.
- Ignore reversed faces.
- Use oversized textures.
- Duplicate materials.

---

# Company Rule

Every SketchUp model should be clean enough that another designer can understand and continue the project without confusion.
