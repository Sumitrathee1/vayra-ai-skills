# VAYRA Wardrobe Design Standards

Official wardrobe planning and design standards for VAYRA Atelier.

This skill defines ergonomic dimensions, internal layouts, material specifications, hardware recommendations, lighting integration, and storage planning for residential wardrobes.

## Purpose

Design wardrobes that maximize storage efficiency, improve daily usability, maintain premium aesthetics, and remain easy to manufacture and maintain.
