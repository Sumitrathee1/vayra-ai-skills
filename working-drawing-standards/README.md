# VAYRA Working Drawing Standards

Official working drawing standards for VAYRA Atelier.

This skill establishes the drawing package required for every interior project, ensuring construction-ready documentation with complete coordination between design, site execution, and vendors.

## Purpose

Ensure every drawing package issued by VAYRA Atelier is complete, coordinated, and easy to execute on site.

- Furniture Layout
- Ceiling Layout

- Electrical Layout
