# Mission

Create accurate, coordinated, and execution-ready drawings that eliminate ambiguity during construction.

---

# Drawing Package

Every project should include:

✓ Cover Sheet

✓ General Notes

✓ Existing Layout

✓ Demolition Plan (if applicable)

✓ Furniture Layout

✓ Ceiling Layout

✓ Flooring Layout

✓ Electrical Layout

✓ Lighting Layout

✓ Plumbing Layout

✓ HVAC Coordination (if applicable)

✓ Wall Elevations

✓ Sections

✓ Detailed Joinery Drawings

✓ Door Schedule

✓ Window Schedule

✓ Material Schedule

✓ Hardware Schedule

---

# Furniture Layout

Must include:

- All furniture dimensions
- Clear circulation
- Furniture codes
- Room names
- North direction
- Levels if required

---

# Ceiling Layout

Show:

- Ceiling profile
- Cove lighting
- Spotlights
- Pendant lights
- AC vents
- Access panels
- Ceiling levels

Coordinate with electrical drawings.

---

# Flooring Layout

Include:

- Tile direction
- Tile size
- Pattern
- Threshold details
- Floor finish codes

---

Switches

Geyser

Decorative Lights

Emergency Lighting (where required)

# Wall Elevations

Include:

Dimensions

Joinery

# Joinery Details

# Drawing Coordination

Verify:

Electrical vs Ceiling

Plumbing vs Furniture

HVAC vs Ceiling

Furniture vs Structure

Door swing vs Furniture

Window vs Wardrobe

Lighting vs Joinery
